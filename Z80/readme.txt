how to compile
--------------

install SDCC 3.5.0 at C:\SDCC\
add C:\SDCC\BIN to path variable

Run compile_app.bat in the directory of the Z80-program you want to compile.
In order to compile the PIC-code you have to install mikroPascal 8.
If you want to compile the emulator (folder E-Z80), you will need Visual Studio 2017.
